== Installation ==

To use Wordpress Signaturer, you will need:

* 	an installed and configured copy of WordPress version 3.0.x.

*	FTP or SFTP access to your web host

= New Installations =

1.	Download the All Posts WordPress MU Widget archive in zip or gzipped tar format and
	extract the files on your computer. 

2.	Create a new directory named `all_posts_mu.php` in the `wp-content/plugins`
	directory of your WordPress installation. Use an FTP or SFTP client to
	upload the contents of your All Posts WordPress MU Widget archive to the new directory
	that you just created on your web host.

3.	Log in to the WordPress Dashboard and activate the All Posts WordPress MU Widget plugin.

5.	Configure the plugin and add it to the plugin settings page.
